How it works (Model B)

  1. You start with 500 coins. Buy player cards before their fixtures.
  2. When a match finishes, the card value changes based on the player's performance.
  3. Profits/losses are unrealized until you hit Sell on a finished card.
  4. Your Net Worth = Balance + Value of all unsold (open) cards.

Scoring → Value

Points add up from events. Then: newValue = base × (1 + score/50) (clamped between 0.2× and 3× base).

  • Team result: Win +10, Draw +3
  • Minutes: <60 +2, 60–89 +4, 90 +6
  • Goals: FWD +8, MID +10, DEF/GK +12 each
  • Assists: +6 each
  • Clean sheet (DEF/GK, ≥60′, team concedes 0): +8
  • GK saves: +1 per 3 saves; Penalty save +8
  • Cards: Yellow −3, Red −7
  • Own goal −6; Missed pen −5; Pen conceded −5

Tips

  • Diversify across fixtures and positions.
  • For GKs/DEFs, clean sheets move value a lot.
  • Use the Fixtures tab to tweak stats for custom scenarios.
